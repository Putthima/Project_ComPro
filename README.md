##### :file_folder: ***Description***
> ***Help-T อุปกรณ์สำหรับการส่งข้อความให้คนรู้ใจเพื่อให้ติดต่อกลับมาและลดโอกาสการเกิดปัญหา รวมถึงส่งสัญญาณแสงและเสียงให้คนรอบตัวเมื่ออยู่ในสถาการณ์คับขัน
